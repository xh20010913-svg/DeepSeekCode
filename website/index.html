DeepSeek 优先的本地 Agent 运行时

DeepSeekCode

面向本地项目开发、办公文档生成和长任务协作的终端 Agent。v0.2.1 公开说明以真实接入能力为准：native tool calling、本地 typed tools、SQLite 持久状态、TencentDB-Agent-Memory 长期记忆、skills/plugins、权限 gate、缓存与费用遥测。

快速开始 阅读中文说明书
要求 native tool calls 无 JSON planner fallback 内置 TencentDB-Agent-Memory 中文默认

Quickstart

在独立项目目录中运行。

源码仓库只用于构建和发布；真实任务、测试产物和运行数据应放在单独的项目目录。

git clone https://github.com/xh20010913-svg/DeepSeekCode.git
cd DeepSeekCode
npm install
npm run build
npm run start -- --project "D:\work\agent-test"

Runtime Loop

主链路使用 native tools 驱动本地执行。

1messages + tools schema
2TDAI memory recall
3DeepSeek tool_calls
4local tools + tool_result
5capture + next turn

Long-Term Memory

长期记忆进入实际运行链路。

DeepSeekCode 内置 TencentDB-Agent-Memory 的 MIT runtime。模型调用前召回相关偏好、项目事实和历史决策；成功回合结束后捕获对话并提取结构化记忆。默认使用本地 SQLite，embedding 和 Tencent Cloud VectorDB 需要显式配置。 

/memory status
/memory search 中文默认
/memory conversation 继续完善

set DEEPSEEKCODE_TDAI_MEMORY=off

Skills / Plugins

Skills 和 plugins 写入 .deepseekcode。

支持本地路径、GitHub URL、Git URL 和 file:// Git 源。可以兼容读取 .claude skill/plugin，但安装目标是 DeepSeekCode 自己的运行目录。

/skills install https://github.com/example/agent-skills/tree/main/office/report
/plugins install file:///D:/repos/deepseekcode-plugin.git#main
/skills validate
/plugins validate

Reports

场景报告用于判断能力和成本。

报告包含模型、token、cache hit/miss、工具次数、产物、失败点和修复建议。

set DEEPSEEKCODE_PROMPT_AUDIT_DIR=D:\work\agent-test\prompt-audit
/runs report latest "D:\work\agent-test"
