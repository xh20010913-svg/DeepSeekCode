INSTALL

全局安装，在当前工作区启动。

从任意仓库或工作目录运行 `deepseekcode`。运行状态、会话、记忆、远程绑定和产物记录会保存在当前工作区的 `.deepseekcode` 中，便于继续任务和恢复上下文。 

npm install -g @xh12312/deepseekcode --registry https://registry.npmjs.org/
cd /d D:\code\DeepSeekTest
deepseekcode --model deepseek-v4-flash

CAPABILITIES

核心能力。

Native tool loop

模型通过 DeepSeek tools 数组请求本地工具；runtime 负责 Zod 校验、权限、执行、tool_result 回放和失败反馈。

Project state

每个工作区独立保存 transcript、runs、events、artifacts、usage、approval gates 和长期记忆。

Remote control

企业微信和个人微信 OpenClaw 都保留。远程端只显示简洁进度、权限审批、旁路问答和产物预览。

Skills / plugins

支持本地、GitHub、Git URL 和 file:// skill/plugin 安装。模型可通过 search_skills 与 invoke_skill 自动调用。

Multi-agent

主 agent 可以创建角色、共享黑板和 Reviewer 验收角色。该能力可测试，但仍标为实验中。

Artifact delivery

网页优先截图，Office/PDF 发可预览文件，多文件项目发摘要、入口和 manifest，不刷屏发源码。

REMOTE

电脑继续运行，微信负责控制和看结果。

推荐在电脑 TUI 中通过 `/remote-control` 绑定个人微信或企业微信。远程消息进入同一个 QueryEngine、同一套权限 gate 和同一份项目状态，不另造孤立 agent。
